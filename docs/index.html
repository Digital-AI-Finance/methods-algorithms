Course Information

Program: MSc Data Science
Sessions: 6 x 3 hours
Prerequisites: Python, Statistics, Linear Algebra
Domain Focus: Finance and Banking

PMSP Framework

Each lecture follows the PMSP framework:

Problem (15 min) - Real finance use case
Method (45 min) - Algorithm theory and mathematics
Solution (45 min) - Implementation and results
Practice (75 min) - Hands-on notebook exercises

Introduction & Linear Regression

OLS, Gradient Descent, Regularization

Learn the fundamentals of linear regression including Ordinary Least Squares, gradient descent optimization, and regularization techniques (Ridge, Lasso). Apply these to house price prediction and factor models in finance.

KNN & K-Means Clustering

Distance Metrics, Clustering, Elbow Method

Explore instance-based learning with K-Nearest Neighbors and unsupervised clustering with K-Means. Learn distance metrics, the curse of dimensionality, and cluster validation. Apply to customer segmentation and anomaly detection.
